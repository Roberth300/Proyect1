Computer Programming

Computer programming is the process of creating programs or applications
through the development of source code which, this source code is based
on a set of instructions that the computer follows to execute a program.

Information technology

Information technology (IT) is a set of related fields that encompass
computer systems, software, programming languages, and data and information
processing and storage. IT is part of information and communications technologies (ICT)..
